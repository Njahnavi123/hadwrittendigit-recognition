# Handwritten Digit Recognition

This project is a simple Handwritten Digit Recognition system developed using Python and Machine Learning concepts.

## Description
The application recognizes handwritten digits (0â€“9) based on numeric pixel values.
It uses a machine learning model to classify digits.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Project Structure
- dataset/ : Contains digit dataset
- model/ : Stores trained model files
- train_model.py : Script to train the model
- app.py : Script to test the model

## How to Run the Project

1. Install required libraries:
   ```bash
   pip install -r requirements.txt
   ```
2. Train the model
   ```bash
   python train_model.py
   ```
3. Run the application
   ```bash
   python app.py
   ```
